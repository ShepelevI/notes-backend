## Массив

Коллекция данных заданного типа **фиксированной** длины, где каждый элемент имеет индекс (начальный индекс - **ноль**), к которому можно обращаться.

Выделение памяти происходит в момент инициализации, хранится в ней как последовательность блоков указанного типа. 

**Доступ O(1), поиск, вставка и стирание O(n)**.

При передаче или присваивании создается **копия** содержимого, для возможности изменения исходного массива, использовать **срез** или передавать **по указателю (*)**.

* создание (указать имя, длину, тип значений):

`var arr [3]int`

* прямая инициализация:

`var arr = [3]int{1, 2, 3}`

* без **var**:

`arr := [3]int{1, 2, 3}`

Обход через **for**:

    for i := 0; i < len(arr); i++ {
        fmt.Println(arr[i])
    }

Обход через **range**:

    for index, value := range arr {
        fmt.Printf("Индекс: %d, Значение: %d\n", index, value)
    }

Выход за границы массива (доступ к индексу за пределами длины) выдаст **ошибку времени выполнения**. 

Всегда проверять находится ли индекс в допустимом диапазоне массива перед доступом к элементам через сравнение индекса и длины массива:

    var arr [5]int
    index := 10 
    if index < len(arr) {
        fmt.Println(arr[index])
    } else {
        fmt.Println("Индекс выходит за границы массива.")
    }

## Срез

Абстракция над массивом (**ссылка на базовый массив без создания копии данных**).

Имеет **динамичную** длину.
