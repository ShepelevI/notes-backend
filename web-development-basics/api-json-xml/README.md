**Сериализация** — процесс преобразования объекта в поток байтов для сохранения или передачи.

Обратная операция - **десериализация** (т.е. восстановление объекта/структуры из последовательности байтов).
Синонимом можно считать термин "маршалинг" (от англ. `marshal` — упорядочивать).

Минусы сериализации - нарушение инкапсуляции, т.е. после сериализации "приватные" свойства структур могут быть доступны
для изменения.

Пример сериализации в Go - преобразование структур в `json` объекты.
Кроме `json` существуют различные кодеки типа `MessagePack`, `CBOR`.
